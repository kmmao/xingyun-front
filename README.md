### 项目介绍

星云 ERP 是基于 SpringBoot 框架的中小企业完全开源的 ERP。

### 环境版本说明

- nodejs v20.17.0
- npm 10.8.2
- pnpm 9.7.1
  > pnpm 安装命令：npm install -g pnpm@9.7.1

### 主要技术框架

- Vue 3.3.4
- ant-design-vue 4.0.7
- vxe-table 4.4.5
- vue-vben-admin
